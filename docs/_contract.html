Variables Contract (expectedVariables)

Writing complex HTML templates with Twig carries a risk: if a variable is omitted in the controller (e.g. user.email or an activation link), Twig will throw a rendering error or display an incomplete email in production.

To solve this, the MailTemplate entity has an expectedVariables field. This contract validates the context array before delivery.

Security check:

If any required variable (including dotted notations for nested structures, e.g. user.firstName) is missing from the $context parameter, the service immediately throws a Callisto\CallistoMailer\Exception\MissingTemplateVariablesException and aborts sending.
