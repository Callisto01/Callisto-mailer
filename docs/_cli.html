CLI Tools

1. Test Email Sending

This command parses the contract, automatically creates fake mock variables context matching the schema, and fires the email to test rendering and deliverability: 

php bin/console callisto:mailer:test-send welcome_user en test@example.com

2. Synchronize Twig Templates to Database

Extract email blocks (subject, layout, content) written as raw physical Twig files in your IDE and sync them to your database: 

php bin/console callisto:mailer:import-templates templates/emails/
