Events

The bundle dispatches two major events via Symfony's EventDispatcher to customize or log email activities:

Event Name Description
Callisto\CallistoMailer\Event\BeforeTemplateMailSendEvent Dispatched right before sending. Allows you to modify the Email object (add CCs, headers) or alter the Twig context variables.
Callisto\CallistoMailer\Event\AfterTemplateMailSendEvent Dispatched immediately after sending successfully. Useful for auditing, history logs, or analytics.
