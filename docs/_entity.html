MailTemplate Entity

Here is the description of the fields in the Callisto\CallistoMailer\Entity\MailTemplate Doctrine entity:

Property Type Description
id int Auto-incremented unique database identifier.
code string Logical unique code identifying the email template (e.g. welcome_user).
locale string ISO-639-1 language code (e.g. fr, en). Defaults to fr.
subject string Email subject, compilable with Twig (e.g. Welcome {{ user.firstName }}!).
layout string Layout alias (tailwind, bootstrap) or absolute Twig path.
content text Email body content written in HTML/Twig.
expectedVariables array JSON array of mandatory variables validation contract (e.g. ["user.firstName", "loginUrl"]).
