Introduction

The callisto/callisto-mailer package is a private, modern Symfony 8 bundle designed to store, compile, and distribute email templates managed in a database. It allows you to edit the subject and body of emails at any time without requiring a code redeployment.

The bundle automatically integrates high-fidelity responsive layouts (supporting Tailwind CSS and Bootstrap guidelines) with inline styling to ensure optimal rendering across all major email clients (Gmail, Outlook, Apple Mail, etc.).

Key Features:

  • Database Storage: Centralized administration for your email templates.
  • Dynamic Twig Compilation: On-the-fly variable parsing in both the subject and the body.
  • Validation Contract (JSON): Secures delivery by verifying the presence of required variables.
  • Native Multi-language (i18n): Built-in translation management with configurable fallbacks.
