Multi-language Support (i18n)

Thanks to the unique constraint on the composite key ['code', 'locale'], you can store multiple translations for the same template code in your database.

Specify the locale using the locale parameter: 

$databaseMailerService->send(
    code: 'welcome_user',
    recipient: 'john.doe@example.com',
    context: ['user' => $user],
    locale: 'en' // Finds template with locale 'en'
);

If the requested locale is missing, the service automatically falls back to the configured default_locale.
