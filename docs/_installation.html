Installation

Since the package is public, you can install it directly via Composer without declaring any custom VCS repositories: 

composer require callisto/callisto-mailer

If you are not using Symfony Flex, manually register the bundle in config/bundles.php:

// config/bundles.php

return [
    // ...
    Callisto\CallistoMailer\CallistoMailerBundle::class => ['all' => true],
];
