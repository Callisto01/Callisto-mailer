Local Loading Error (CORS)

The browser blocked loading local HTML documentation files due to CORS security restrictions associated with the file:// protocol.

To view the site with its imported HTML sub-files, please start a simple local web server. For example, run the following command in the /docs directory: 

php -S localhost:8000

Then open http://localhost:8000 in your browser.
