Attachments

The $attachments argument accepts an array containing:

  • Absolute physical file paths as strings.
  • Instances of Symfony\Component\Mime\Part\DataPart for dynamically generated files.
use Symfony\Component\Mime\Part\DataPart;

$databaseMailerService->send(
    code: 'invoice_template',
    recipient: 'client@example.com',
    context: ['order' => $order],
    attachments: [
        '/var/www/app/public/pdf/invoice-123.pdf', // Physical file path
        new DataPart($csvContent, 'report.csv', 'text/csv') // Dynamic content
    ]
);
