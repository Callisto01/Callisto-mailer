Layouts

The package provides two high-fidelity graphic structures (layouts) ready for use:

  • tailwind: Modern clean layout with an indigo-style button (.btn-indigo).
  • bootstrap: Classic sober style with standard buttons (.btn-primary).

Overriding Layouts:

To override the default designs of these layouts without altering your database entities, simply create the following templates in your main application:

  • templates/bundles/CallistoMailerBundle/layouts/base_tailwind.html.twig
  • templates/bundles/CallistoMailerBundle/layouts/base_bootstrap.html.twig
