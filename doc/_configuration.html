Basic Configuration

Create the configuration file in your main application at config/packages/callisto_mailer.yaml: 

# config/packages/callisto_mailer.yaml
callisto_mailer:
    # Default language used if the requested translated template is missing
    default_locale: 'en'
    
    # Configure available layouts
    layouts:
        tailwind: '@CallistoMailer/layouts/base_tailwind.html.twig'
        bootstrap: '@CallistoMailer/layouts/base_bootstrap.html.twig'
